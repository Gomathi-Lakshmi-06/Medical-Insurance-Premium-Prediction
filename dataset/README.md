# Dataset

This project uses the Medical Insurance Cost Prediction dataset.

File:

* insurance.csv

Dataset Features:

* age: Age of the customer
* sex: Gender of the customer
* bmi: Body Mass Index
* children: Number of dependent children
* smoker: Smoking status (Yes/No)
* region: Residential region
* charges: Medical insurance premium charges

Purpose:
The dataset is used to analyze factors affecting insurance premiums and to build a Machine Learning model for premium prediction.
