# Notebooks

This folder contains Databricks notebooks used in the Medical Insurance Premium Prediction project.

Notebooks Included:

* Data Loading and Exploration
* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Random Forest Model Training
* Feature Importance Analysis
* Premium Prediction
* Model Evaluation

Technologies Used:

* Python
* PySpark
* Databricks
* SQL
* Matplotlib

Purpose:
These notebooks demonstrate the complete Machine Learning workflow from data preparation to insurance premium prediction and performance evaluation.
