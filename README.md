# Medical Insurance Premium Prediction using Machine Learning

## Project Overview

This project predicts medical insurance premiums based on customer information such as age, gender, BMI, number of children, smoking status, and region. The goal is to identify the key factors influencing insurance charges and build an accurate machine learning model for premium prediction.

---

## Problem Statement

Insurance companies need reliable methods to estimate medical insurance premiums. This project uses Machine Learning techniques to analyze customer attributes and predict insurance charges accurately.

---

## Dataset Information

Dataset: **Medical Insurance Cost Prediction Dataset**

Features:

* Age
* Sex
* BMI
* Children
* Smoker
* Region
* Charges (Target Variable)

## Model Performance

| Metric   | Value   |
| -------- | ------- |
| RMSE     | 4060.46 |
| MAE      | 2430.49 |
| R² Score | 0.8761  |

---

## Key Findings

* Smoking status is the most important factor affecting insurance premiums.
* Age and BMI significantly influence premium charges.
* Random Forest Regression achieved strong predictive performance.
* The model explains approximately 87.6% of the variation in insurance charges.
